## 🚀 Cos'è AssemblyONLINE?

AssemblyOnline è una **webapp open source** che emula un ambiente x86/8086 direttamente nel browser, senza installare nulla. Scrivi codice Assembly, eseguilo, e osserva registri, memoria e output in tempo reale — come emu8086, ma accessibile ovunque.

### ✨ Funzionalità principali

- 📝 **Editor integrato** con syntax highlighting per Assembly x86/8086
- ⚙️ **Esecuzione passo-passo** con visualizzazione dei registri in tempo reale
- 🧠 **Memoria e stack** ispezionabili durante l'esecuzione
- 🌐 **Zero installazione** — gira interamente nel browser
- 🔀 **Funzione Remix** disponibile direttamente su Google AI Studio
